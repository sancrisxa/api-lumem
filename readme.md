# Lumen PHP Framework



## Intrução para usar o projeto

- git clone
- composer intall
- php -S localhost:8080 -t public
- php artisan migrate
- php artisan db:seed
- criar o arquivo .env usado o .env.example


Irei fazer o dump da database do projeto. Mas as migrations e as seeders já irão criar a base.
